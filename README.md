## Run Locally

Clone the project

```bash
  https://github.com/pbbhattpriyanshu/Java-DSA.git
```

Go to the project directory

```bash
  cd Java-DSA
```

Run Java files

You can compile and run any .java file using:

```bash
  javac filename.java
  java filename
```
